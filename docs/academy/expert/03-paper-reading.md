## How to Read AI Security Papers

```
1. Abstract ‚Üí Problem + Solution (2 min)
2. Intro ‚Üí Context + Motivation (5 min)
3. Threat Model ‚Üí Attack assumptions (5 min)
4. Methodology ‚Üí How it works (10 min)
5. Results ‚Üí What they achieved (5 min)
6. Limitations ‚Üí What doesn't work (3 min)
```
