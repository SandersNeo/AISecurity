# STRIKE Red Team Labs

> **Lab Track: Offensive Security Exercises**

---

## Overview

The Red Team lab track focuses on offensive skills: attack crafting, exploitation, evasion, and campaign execution using STRIKE and related tools.

---

## ⚠️ Ethical Guidelines

**These exercises are for authorized testing only:**
- Only target systems you own or have permission to test
- Follow all applicable laws and regulations
- Practice responsible disclosure
- Document all activities

---

## Prerequisites

- Completed Modules 01-04
- Understanding of attack techniques
- STRIKE tools installed
- Access to lab sandboxes

---

## Labs

### Lab 001: STRIKE Setup
**Time:** 30 minutes | **Difficulty:** Beginner

Install and configure STRIKE:
- Tool installation
- Payload libraries
- Target configuration
- Safety settings

### Lab 002: Basic Prompt Injection
**Time:** 40 minutes | **Difficulty:** Beginner

Craft your first injections:
- Direct override attempts
- Instruction manipulation
- Context exploitation
- Success verification

### Lab 003: Jailbreak Techniques
**Time:** 45 minutes | **Difficulty:** Intermediate

Bypass safety measures:
- DAN-style attacks
- Persona manipulation
- Gradual escalation
- Effectiveness measurement

### Lab 004: Defense Evasion
**Time:** 50 minutes | **Difficulty:** Intermediate

Bypass known defenses:
- Pattern evasion
- Encoding techniques
- Semantic manipulation
- Detection testing

### Lab 005: Payload Mutation
**Time:** 50 minutes | **Difficulty:** Intermediate

Generate attack variants:
- Mutation strategies
- Grammar-based fuzzing
- Coverage analysis
- Effectiveness tracking

### Lab 006: Automated Campaigns
**Time:** 60 minutes | **Difficulty:** Advanced

Run structured attack campaigns:
- Campaign planning
- Target mapping
- Attack execution
- Results analysis

### Lab 007: Tool Injection
**Time:** 60 minutes | **Difficulty:** Advanced

Attack agentic systems:
- MCP exploitation
- Tool manipulation
- Privilege escalation
- Multi-hop attacks

### Lab 008: Novel Attack Research
**Time:** 90 minutes | **Difficulty:** Expert

Discover new techniques:
- Research methodology
- Variant generation
- Validation testing
- Responsible documentation

## Getting Started

Start with Lab 001:

```bash
python -m labs.red.lab_001
```
