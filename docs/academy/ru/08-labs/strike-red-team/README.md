## Лаборатории

### [Lab 001: Базовая инъекция промптов](lab-001-basic-injection.md)
- 5 категорий атак: переопределение, извлечение, ролевая игра, разделители, кодирование
- Атака на уязвимый TargetChatbot
- Оценка на основе баллов (100 очков)

### [Lab 002: Непрямая инъекция промптов](lab-002-indirect-injection.md)
- Техники отравления RAG
- Скрытые инструкции в документах
- Многошаговые цепочки атак

### [Lab 003: Техники джейлбрейков](lab-003-jailbreak-techniques.md)
- DAN и персонажные атаки
- Манипуляция контекстом
- Методы обхода через кодирование

### [Lab 004: Атаки на агентов](lab-004-agent-attacks.md)
- Атаки злоупотребления инструментами
- Путаница мульти-агентов
- Эскалация привилегий
