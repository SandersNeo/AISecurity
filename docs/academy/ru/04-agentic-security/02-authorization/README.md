# Авторизация

> **Модуль 04.2: Паттерны авторизации**

---

## Обзор

Авторизация агентов определяет, какие действия агент может выполнять и к каким ресурсам имеет доступ.

---

## Уроки

### [01. Паттерны авторизации](01-authorization-patterns.md)
- RBAC для агентов
- Токены возможностей
- Модели разрешений
- Гранулярный контроль доступа

---

## Ключевые паттерны

| Паттерн | Описание | Применение |
|---------|----------|------------|
| **RBAC** | Доступ по ролям | Статические разрешения |
| **Capability** | Токены на действия | Динамические права |
| **ABAC** | Доступ по атрибутам | Контекстные решения |
