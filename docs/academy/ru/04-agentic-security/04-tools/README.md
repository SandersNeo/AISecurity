## Темы

- Безопасность определения инструментов
- Валидация аргументов
- Песочница выполнения
- Санитизация результатов

---

## Модель безопасности инструментов

```
Вызов инструмента:
├── Валидация ввода → Проверка схемы, санитизация
├── Авторизация → Проверка возможности
├── Выполнение → Изолированная среда
└── Вывод → Санитизация результата
```

## Общие угрозы

| Угроза | Атака | Защита |
|--------|-------|--------|
| Инъекция | Некорректные аргументы | Валидация схемы |
| Побег | Инъекция команд | Песочница |
| Эксфильтрация | Утечка данных через инструменты | Фильтрация вывода |
