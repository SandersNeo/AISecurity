# Архитектуры

> **Модуль 04.1: Паттерны архитектуры агентов**

---

## Обзор

Архитектуры агентов определяют, как AI-агенты организованы, взаимодействуют с инструментами и управляют состоянием.

---

## Уроки

- Паттерн ReAct
- Паттерны использования инструментов
- Оркестрация мульти-агентов
- Управление памятью
- Архитектура First-Security

---

## Паттерны архитектуры

```
Один агент:          Мульти-агентная:
    LLM                Оркестратор
     │                  /   |   \
  Инструменты     Агент1 Агент2 Агент3
                      │      │      │
                 Инстр.  Инстр.  Инстр.
```

---

## Соображения безопасности

| Компонент | Угроза | Защита |
|-----------|--------|--------|
| LLM | Инъекция промптов | Валидация ввода |
| Инструменты | Злоупотребление привилегиями | Лимиты возможностей |
| Память | Отравление | Валидация состояния |
| Оркестрация | Перехват | Границы доверия |
