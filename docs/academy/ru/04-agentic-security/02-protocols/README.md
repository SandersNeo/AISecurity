# Протоколы

> **Модуль 04.2: Безопасность протоколов коммуникации агентов**

---

## Обзор

Протоколы агентов определяют, как агенты общаются друг с другом и с внешними системами.

---

## Уроки

- MCP (Model Context Protocol)
- A2A (Agent-to-Agent)
- Безопасность Function Calling
- Делегирование доверия

---

## Безопасность протоколов

| Протокол | Применение | Фокус безопасности |
|----------|------------|---------------------|
| **MCP** | Интеграция инструментов | Валидация схемы |
| **A2A** | Мульти-агентный | Подписание сообщений |
| **Function Calling** | LLM→Инструменты | Валидация аргументов |

---
