# Циклы агентов

> **Модуль 04.1: Безопасность циклов агентов**

---

## Обзор

Циклы агентов — итеративные процессы, где AI принимает решения и выполняет действия. Защита циклов критична для предотвращения неконтролируемого поведения.

---

## Уроки

### [01. Безопасность циклов агентов](01-agent-loop-security.md)
- Предотвращение перехвата циклов
- Лимиты итераций
- Валидация состояния
- Обнаружение бесконечных циклов

---

## Ключевые риски

| Риск | Описание | Митигация |
|------|----------|-----------|
| **Перехват цикла** | Атакующий направляет итерации | Валидация состояния |
| **Бесконечный цикл** | Неконтролируемые итерации | Лимиты и таймауты |
| **Накопление ошибок** | Ошибки усиливаются с итерациями | Проверки качества |
