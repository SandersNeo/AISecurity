# Безопасность инструментов

> **Модуль 04: Безопасность инструментов**

---

## Обзор

Безопасность инструментов охватывает защиту функций, которые AI-агенты могут вызывать для взаимодействия с внешними системами.

---

## Уроки

### [01. Безопасность инструментов](01-tool-security.md)
- Валидация инструментов
- Управление разрешениями
- Санитизация вывода
- Песочница выполнения

---

## Ключевые риски

| Риск | Атака | Защита |
|------|-------|--------|
| **Инъекция** | Некорректные аргументы | Валидация схемы |
| **Побег** | Инъекция команд | Песочница |
| **Эксфильтрация** | Утечка через инструменты | Фильтрация вывода |
