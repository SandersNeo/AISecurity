# Атаки на агентов

> **Модуль 03.1: Атаки на AI-агентов**

---

## Обзор

AI-агенты расширяют возможности LLM инструментами и автономным выполнением задач, создавая новые векторы атак.

---

## Уроки

### [02. ASI02 Эскалация привилегий](02-ASI02-privilege-escalation.md)
- Инъекция ролей
- Обход разрешений
- Манипуляция контекстом
- Атаки "запутанного заместителя"

---

## Ключевые риски

| Атака | Описание | Серьёзность |
|-------|----------|-------------|
| **Эскалация привилегий** | Агент получает больше прав | Критическая |
| **Цепочка инструментов** | Эксплуатация комбинаций | Высокая |
| **Confused Deputy** | Агент злоупотребляет правами | Высокая |
