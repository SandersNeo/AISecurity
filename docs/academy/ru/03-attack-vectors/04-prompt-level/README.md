# Prompt-Level техники атак

> **Подмодуль 03.4: Продвинутые эксплойты через промпт**

---

## Обзор

Помимо базовых инъекций и джейлбрейков, продвинутые атаки на уровне промпта используют сложные техники для манипуляции поведением модели. Эти методы часто комбинируют несколько подходов и требуют глубокого понимания того, как модели обрабатывают контекст.

---

## Категории техник

| Техника | Механизм | Сложность детекции |
|---------|----------|-------------------|
| **Virtualization** | Создание вымышленных фреймов | Высокая |
| **Payload Smuggling** | Скрытие инструкций в контенте | Очень высокая |
| **Context Manipulation** | Переосмысление разговора | Высокая |
| **Multi-Modal** | Cross-modality атаки | Очень высокая |

---

## Уроки

### 01. Virtualization атаки
**Время:** 40 минут | **Сложность:** Высокая

Создание атак в вымышленных фреймах:
- Создание гипотетических сценариев
- Эксплуатация образовательного контекста
- Вложенные контексты реальности
- Обход защит через абстракцию

### 02. Payload Smuggling
**Время:** 45 минут | **Сложность:** Экспертная

Скрытие вредоносных инструкций:
- Схемы кодирования (Base64, ROT13)
- Unicode и невидимые символы
- Эксплуатация format strings
- Стеганографические техники

### 03. Context Manipulation
**Время:** 40 минут | **Сложность:** Высокая

Переосмысление разговоров:
- Authority injection
- Манипуляция историей разговора
- Эксплуатация trust контекста
- Извлечение system prompt через framing

### 04. Cross-Modal атаки
**Время:** 45 минут | **Сложность:** Экспертная

Эксплуатация vision и других модальностей:
- Скрытый текст в изображениях
- Аудио стеганография
- Multi-modal context confusion
- Эксплуатация границ модальностей

## Ключевой инсайт

Продвинутые атаки часто комбинируют:
1. **Framing** — создать приемлемый контекст
2. **Smuggling** — скрыть payload
3. **Persistence** — удержать через несколько turn'ов
4. **Evasion** — обойти детекцию

---

## Сложности детекции

| Атака | Почему сложно обнаружить |
|-------|-------------------------|
| Virtualization | Легитимный вымышленный контент существует |
| Smuggling | Закодированный контент — валидные данные |
| Context Manipulation | Тонкий reframing выглядит естественно |
| Multi-modal | Требуется cross-modal анализ |
