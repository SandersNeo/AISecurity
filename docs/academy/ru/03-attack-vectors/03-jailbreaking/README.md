## Обзор

Джейлбрейкинг — техники обхода ограничений безопасности LLM через манипуляцию промптами, ролевые игры и специальные кодировки.

---

## Уроки

### [01. Техники джейлбрейков](01-jailbreak-techniques.md)
- Таксономия атак
- Ролевые атаки (DAN, персонажи)
- Атаки кодирования (Base64, ROT13)
- Манипуляция инструкциями

### [02. Мультиязычные атаки](02-multilingual-attacks.md)
- Языковой разрыв в safety-обучении

- Смешение скриптов

- Многоуровневая защита
