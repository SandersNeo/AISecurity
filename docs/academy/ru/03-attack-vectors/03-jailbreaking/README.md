# Продвинутый Jailbreaking

> **Подмодуль 03.3: Экспертные техники обхода**

---

## Обзор

Этот подмодуль охватывает продвинутые техники jailbreaking, которые используют экспертные исследователи. Требует глубокого понимания работы моделей.

---

## Продвинутые техники

| Техника | Сложность | Эффективность |
|---------|-----------|---------------|
| **Semantic reframing** | Высокая | Высокая |
| **Nested contexts** | Очень высокая | Высокая |
| **Token-level** | Экспертная | Средняя |
| **Training data** | Экспертная | Переменная |

---

## Уроки

### 01. Семантическое переосмысление
**Время:** 45 минут | **Сложность:** Высокая

Эксплуатация смысла:
- Цепочки эвфемизмов
- Переопределение контекста
- Frame shifting
- Семантическая неоднозначность

### 02. Вложенные контексты
**Время:** 45 минут | **Сложность:** Экспертная

Слои внутри слоёв:
- Fiction within fiction
- Meta-level manipulation
- Reality boundary blur
- Рекурсивные контексты

### 03. Token-level эксплойты
**Время:** 50 минут | **Сложность:** Экспертная

Exploit токенизации:
- Token boundary manipulation
- Rare token sequences
- Encoding tricks
- Format string style

---

## Ladder сложности

```
Уровень 1: Прямые инструкции     → Заблокировано
Уровень 2: Persona, encoding     → Частично блокируется
Уровень 3: Multi-turn, crescendo → Обнаруживается
Уровень 4: Semantic reframing    → Сложно обнаружить
Уровень 5: Token-level           → Передний край
```
