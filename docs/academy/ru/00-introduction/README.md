## Прежде чем начать

### Предварительные требования

Не требуется предварительных знаний по AI security, но вы должны иметь:

- **Опыт программирования** — Уверенное чтение/написание Python
- **Базовые ML концепции** — Понимание того, что делают нейронные сети (высокий уровень)
- **Security mindset** — Любопытство о том, как системы могут быть атакованы/защищены

### Настройка окружения

Быстрый старт для вашей учебной среды:

```bash
# Клонируйте репозиторий
git clone https://github.com/DmitrL-dev/AISecurity.git
cd AISecurity/sentinel-community

# Создайте virtual environment
python -m venv ai-security-env
source ai-security-env/bin/activate  # Linux/Mac
# или
.\ai-security-env\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt
pip install -e .

# Проверьте
python -c "from sentinel import configure; print('Ready!')"
```

Выберите в зависимости от ваших целей:

Фокус на детекции, мониторинге и реагировании.

```
00-Введение > 01-Fundamentals > 03-Атаки > 06-Продвинутый > 08-Labs
```

Полная учебная программа для полной экспертизы AI security.
```
Все модули последовательно (00 > 08)
```
