# Guardrails

> **Модуль 05.2: Защитные барьеры для входных/выходных данных**

---

## Обзор

Guardrails — защитные барьеры для входных и выходных данных LLM-систем.

---

## Уроки

### [01. Фильтрация ввода](01-input-filtering.md)
- Валидация размера и формата
- Нормализация символов (NFKC, омоглифы)
- Обнаружение паттернов инъекций
- Обнаружение закодированного контента

### [02. Фильтрация вывода](02-output-filtering.md)
- Классификация контента (вредный, успех джейлбрейка)
- Обнаружение PII (email, телефон, SSN, кредитные карты)
- Обнаружение секретов (API-ключи, токены)
- Санитизация ответов

### [03. Фреймворки Guardrails](03-guardrails-frameworks.md)
- NeMo Guardrails
- LlamaGuard
- Rebuff
- Реализация кастомных guardrails
