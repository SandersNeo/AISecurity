## Основные теоремы

### Detection Bounds
```
Теорема (Detection Lower Bound):
Для любого детектора D с false positive rate α,
detection rate β удовлетворяет:

β ≤ 1 - (1-α) · exp(-n · D_KL(P_attack || P_benign))

где n — размер выборки и D_KL — KL divergence.
```

### Robustness Certification
```
Теорема (Lipschitz Robustness):
Для L-Lipschitz классификатора f и входа x,
если ||x - x'|| < ε, тогда:

|f(x) - f(x')| < L · ε

Следствие: Сертифицированный радиус r = margin / L
```

### Game Equilibrium
```
Теорема (Minimax Defense):
Оптимальная стратегия защиты d* удовлетворяет:

max_a min_d L(a, d) = min_d max_a L(a, d) = v*

где v* — значение игры безопасности.
```
