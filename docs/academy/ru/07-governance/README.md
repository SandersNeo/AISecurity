---

## Типы политик

| Тип | Назначение | Применение |
|-----|------------|------------|
| **Доступ** | Кто может делать что | До запроса |
| **Контент** | Что разрешено | До/После |
| **Поведение** | Разрешённые паттерны | В процессе |
| **Соответствие** | Регуляторные требования | Всегда |
