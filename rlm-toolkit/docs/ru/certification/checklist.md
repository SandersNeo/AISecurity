## Уровень 1: Основы

### Установка и настройка
- [ ] Установить `rlm-toolkit` через pip
- [ ] Настроить API ключ для LLM провайдера
- [ ] Выполнить первый RLM запрос

### Базовые концепции
- [ ] Объяснить что такое RLM (Recursive Language Model)
- [ ] Понимать context window vs бесконечный контекст RLM
- [ ] Описать 4-уровневую архитектуру H-MEM

### Базовое использование
- [ ] Создать RLM с OpenAI/Ollama
- [ ] Использовать HierarchicalMemory для персистентности
- [ ] Построить простой RAG pipeline

## Уровень 2: Практик

### MCP интеграция
- [ ] Настроить MCP Server для IDE
- [ ] Использовать все 10 MCP инструментов
- [ ] Установить и настроить VS Code Extension
- [ ] Отслеживать экономию токенов через session stats

### C³ Crystal
- [ ] Объяснить типы Primitive (FUNCTION, CLASS, и т.д.)
- [ ] Использовать HPEExtractor для анализа кода
- [ ] Навигировать результаты CrystalIndexer

## Практическое задание

### Задание 1: Настройка (10 мин)
1. Установить RLM-Toolkit с MCP
2. Настроить для VS Code
3. Индексировать пример проекта

### Задание 2: Анализ (15 мин)
1. Использовать rlm_analyze для security_audit
2. Найти потенциальные проблемы
3. Задокументировать находки

### Задание 3: Интеграция (20 мин)
1. Построить RAG pipeline с H-MEM
2. Отследить и отчитаться по экономии токенов
3. Валидировать freshness индекса
