## Практическая оценка

### Задание 1: Установка (10 мин)
1. Установить RLM-Toolkit с MCP
2. Настроить для VS Code с Extension v2.1.0
3. Запустить `rlm_discover_project` на тестовом проекте

### Задание 2: Memory Bridge (15 мин)
1. Добавить иерархические факты уровней L1 и L2
2. Использовать `rlm_enterprise_context` с semantic routing
3. Установить Git hook и проверить извлечение на коммите

### Задание 3: Анализ (15 мин)
1. Использовать `rlm_analyze` с целью `security_audit`
2. Проверить статус `rlm_health_check`
3. Документировать находки через `rlm_record_causal_decision`

### Задание 4: Интеграция (20 мин)
1. Построить RAG пайплайн с Memory Bridge персистентностью
2. Отследить и отчитаться об экономии токенов (цель: 50x+)
3. Проверить актуальность индекса через dashboard
