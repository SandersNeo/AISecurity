## Course Structure

### SSA (20-25 hours)

| Day | Modules | Focus |
|-----|---------|-------|
| 1 | 0-1 | Concepts, Attacks |
| 2 | 2-3 | Architecture, Installation |
| 3 | 4-5 | Rules, Integration |
| 4 | 5B | CLI Reference |
| 5 | Labs + Exam | Practical + Assessment |

### SSP (80-100 hours)

| Week | Modules | Focus |
|------|---------|-------|
| 1 | 6-7 | Guards, Protocols |
| 2 | 7B-8 | Extended Protocols, HA |
| 3 | 9-10 | Monitoring, Enterprise |
| 4 | Labs + Exam | Practical + Assessment |

### SSE (160-180 hours)

| Week | Modules | Focus |
|------|---------|-------|
| 1-2 | 11-12 | Internals, Custom Guards |
| 3-4 | 13-14 | Plugins, Performance |
| 5-6 | 15-16 | Capstone, Policy/eBPF |
| 7-8 | Project | Final Project |

---

## Teaching Methods

### Lectures
- Use SLIDES.md for presentations
- 30% theory, 70% demos

### Labs
- Students do labs individually
- Instructor reviews in real-time
- Use LABS.md for exercises

### Assessment
- EXAM_BANK.md for quizzes
- Practical labs for certification

## Prerequisites Check

Before each level:

| Level | Student Should Know |
|-------|---------------------|
| SSA | C basics, HTTP, CLI |
| SSP | SSA + Networks, Security |
| SSE | SSP + OS internals |

## Common Issues

| Issue | Solution |
